# 工作流

## 开发流程

1. 接收任务 → 确认需求与约束
2. 编码前门禁 → 阅读规范文档，制定实现计划
3. 实现 → 紧扣范围，避免无关重构
4. 验证 → 测试通过后再提交
5. 文档同步 → 更新相关文档
6. 学习复盘 → 沉淀可复用经验

## 质量门禁

- lint/type 检查通过
- 核心 API 流程验证
- 文档已同步更新

## 视频任务执行

1. 创建任务 → 获取 task_id
2. 轮询状态 → query/video_generation
3. 成功后下载 → files/retrieve 获取 download_url
4. 保存到 works/video/
